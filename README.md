# STEM-Educ8r

Professional landing page showcasing 20 years of STEM education leadership and innovation

## About

This landing page showcases the professional journey, achievements, and expertise of Alex Jones, M.Ed., a distinguished STEM education leader with experience spanning classroom instruction, district-level program management, and state education initiatives.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Interactive Elements**:
  - Testimonial carousel with 5 featured recommendations
  - Modal popup gallery for media features
  - Smooth scroll animations
  - Dynamic navigation
- **Comprehensive Sections**:
  - Professional journey timeline
  - Awards & recognition
  - Certifications (Texas & Arkansas)
  - Impact metrics
  - Featured media appearances
  - Downloadable credentials

## Technologies Used

- HTML5
- CSS3 (Custom animations, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (DM Serif Display, Red Hat Display)
